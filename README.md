# 🔍 Email OSINT Tool

An open-source OSINT (Open Source Intelligence) tool that collects publicly available information related to an email address.

---

## Features

- ✅ Email format validation  
- 🌐 Domain & IP lookup  
- 📡 Mail server (MX) info  
- 🖼️ Gravatar profile detection  
- 🔍 Social media username search  
- 🔐 Data breach check (public sources)  

---

## Tech Stack

- Python 3  
- requests  
- socket  
- hashlib
